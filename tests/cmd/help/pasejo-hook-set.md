```
$ pasejo hook help set
Set hook commands to a store or globally to all stores

Usage: pasejo hook set [OPTIONS]

Options:
  -s, --store <STORE>  Optional name of store to use. Defaults to the default store or the first one defined in the local user configuration
  -g, --global         Add commands to the global configuration or save them per store
      --pull <PULL>    The pull command(s) to set
      --push <PUSH>    The push command(s) to set
      --prepend        Prepend commands instead of replacing them
      --append         Append commands instead of replacing them
  -v, --verbose...     Increase logging verbosity
  -q, --quiet...       Decrease logging verbosity
  -h, --help           Print help

```
