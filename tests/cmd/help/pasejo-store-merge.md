```
$ pasejo store help merge
Merge two stores

Usage: pasejo store merge [OPTIONS] --common-ancestor <COMMON_ANCESTOR> --current-version <CURRENT_VERSION> --other-version <OTHER_VERSION>

Options:
  -s, --store <STORE>
          Optional name of store to use. Defaults to the default store or the first one defined in the local user configuration
      --common-ancestor <COMMON_ANCESTOR>
          The path to the common ancestor of the two stores
      --current-version <CURRENT_VERSION>
          The path to current version of the store
      --other-version <OTHER_VERSION>
          The path to the other version of the store
  -v, --verbose...
          Increase logging verbosity
  -q, --quiet...
          Decrease logging verbosity
  -h, --help
          Print help

```
