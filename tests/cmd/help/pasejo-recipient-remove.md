```
$ pasejo recipient help remove
Remove a recipient

Usage: pasejo recipient remove [OPTIONS] <PUBLIC_KEY>

Arguments:
  <PUBLIC_KEY>  The public key of the recipient to remove

Options:
  -s, --store <STORE>   Optional name of store to use. Defaults to the default store or the first one defined in the local user configuration
  -i, --ignore-unknown  Don't fail on unknown recipients
  -v, --verbose...      Increase logging verbosity
  -q, --quiet...        Decrease logging verbosity
  -h, --help            Print help
```
