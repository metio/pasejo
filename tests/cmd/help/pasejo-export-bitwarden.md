```
$ pasejo export help bitwarden
Export all passwords of a store in Bitwarden JSON format

Usage: pasejo export bitwarden [OPTIONS]

Options:
  -s, --store <STORE>
          Optional name of store to use. Defaults to the default store or the first one defined in the local user configuration
      --organization-id <ORGANIZATION_ID>
          The organization ID to use. When set, outputs organization JSON format
      --collection-id <COLLECTION_ID>
          The collection ID to use
      --collection-name <COLLECTION_NAME>
          The collection name to use
      --username-keys <USERNAME_KEYS>
          [default: login email username]
      --uri-keys <URI_KEYS>
          [default: uri url link site]
      --pretty [<PRETTY>]
          Toggle whether to print pretty JSON or not [default: false] [possible values: true, false]
  -v, --verbose...
          Increase logging verbosity
  -q, --quiet...
          Decrease logging verbosity
  -h, --help
          Print help

```
